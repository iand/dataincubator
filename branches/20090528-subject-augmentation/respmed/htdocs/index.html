Respiratory Medicine Linked Open Data Demonstrator

This service is a demonstration of how data sourced from a Highwire SGML feed from European Respiratory Journal can be published on the web according to linked data principles. The data can be requested as RDF/XML or JSON, and URIs from the dataset can be fed into services such as tabulator or marbles for a more human-readable view

Some starting points:

  • Start browsing from journals
  • Start browsing from keywords
  • An article from the journal Science

Requesting data in different formats:

This is done by appending the querystring parameter 'output' with a value of either 'json' or 'rdf'.

  • An article as RDF/XML (default)
  • An article as JSON
